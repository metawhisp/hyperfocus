## Data plumbing needed (beyond today's Session fields)
Today `Session` has: mission, plannedDurationSeconds, activeFocusSeconds, pausedSeconds, breakCount,
longestStreakSeconds, completionStatus, startedAt, endedAt, intensity, cameraEnabled. To cover all 100:
- **awayCount** per session (for Nine Lives; breakCount ≈ this — verify semantics)
- **nudgeSurvived** flag (Nudge-Proof #57)
- **focusSoundUsed** + mode (Sound & Vision #99)
- **startLatencySeconds** from orb-click to countdown (Speed Demon #95)
- **wandSuggested** flag (Wingman #10, Continuation #85)
Everything else is derivable from the existing fields + the full session history.
